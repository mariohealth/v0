These are example JSONs that our API will return as you query different endpoints:

- `categories.json`:
	- the JSON returned by the `GET /api/categories` endpoint
	- it lists all the categories covered by Mario Health


- `imaging_families.json`:
	- the JSON returned by the `GET /api/families` endpoint with the `imaging` parameter
	- it lists all families in the Imaging category of procedures
-  `surgery_families.json`:
	- same as just above
	- the JSON returned by the `GET /api/families` endpoint with the `surgery` parameter
	- it lists all families in the Surgery category of procedures


- `joint_replacement_procedures.json`:
	- the JSON returned by the `GET /api/procedures` endpoint with the `joint_replacement` parameter
	- it lists all procedures in the Joint Replacement family of procedures
- `mri_scans_procedures.json`:
	- same as just above
	- the JSON returned by the `GET /api/procedures` endpoint with the `mri_scans` parameter
	- it lists all procedures in the MRI Scans family of procedures
